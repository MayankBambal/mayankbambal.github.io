Customer Churn Forecasting Pipeline

Role: Data Science Intern at CIBC USA

Timeline: Jan 2025 – May 2025

Overview

Developed an end-to-end forecasting solution to predict customer churn using a Random Forest classifier. Automated data ingestion and feature engineering to support weekly scoring of over 1 million transaction records.

Technologies

  • Python, Pandas, Scikit-learn
  • Spark MLlib for distributed feature processing
  • Azure Data Factory for pipeline orchestration

Key Results

  • Achieved 85% prediction accuracy on hold-out test set
  • Automated ETL pipeline processed 1M+ records weekly, eliminating manual steps
