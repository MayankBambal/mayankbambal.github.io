Project Overview

Developed a comprehensive predictive model to identify hosts at risk of leaving the platform and inform targeted retention strategies. The project utilized ensemble modeling techniques to achieve high accuracy in predicting host attrition.

Key Features

  • Ensemble model combining Random Forest, Gradient Boosting, and MLP
  • Feature engineering and selection for optimal prediction
  • Model interpretability using SHAP values
  • Actionable insights for retention strategy

Technologies Used

  • Python
  • Scikit-learn
  • XGBoost
  • TensorFlow
  • SHAP
  • Pandas & NumPy
