Project Overview

Created an interactive Tableau dashboard to visualize and analyze crime patterns across Los Angeles. The project provides insights into crime trends, hotspots, and temporal patterns to inform public safety initiatives.

Key Features

  • Interactive crime map with filtering capabilities
  • Temporal analysis of crime patterns
  • Crime type categorization and analysis
  • Custom dashboard for law enforcement insights

Technologies Used

  • Tableau
  • SQL
  • Data Visualization
  • Geospatial Analysis
  • Statistical Analysis
