Automated A/B Testing Pipeline

Role: Data Scientist at Accordion India

Timeline: Sep 2022 – Jul 2024

Overview

Built a fully automated analytics pipeline for A/B tests across web, mobile, and in-store channels. Leveraged AWS services to ingest, process, and analyze experiment data, and surfaced insights in Tableau dashboards.

Technologies

  • AWS S3, Glue, EMR, Step Functions, Lambda
  • Athena for querying, Redshift for warehousing
  • Python (SciPy, statsmodels), Tableau

Key Results

  • Reduced analysis time by 60%
  • Enabled near-real-time decision-making on campaign variants
